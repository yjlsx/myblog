我不想把这本书读成普通的毛泽东早年传记。它真正有意思的地方，是把问题从“毛泽东本人是谁”挪到“毛泽东这个政治形象是怎样被看见、误认、翻译、传播和重塑的”。

读下来我最想留下的是一种史料意识：后来被我们以为清清楚楚的形象，在当时可能只是照片、剪报、传闻、译文和政治宣传拼出来的结果。《红星》之所以重要，不只是因为它报道了陕北，而是因为它让一个原本模糊的政治人物第一次变得可见。

## 一、全书一句话与阅读地图

### 一句话概括

这本书不是普通毛泽东传，也不是单纯研究《红星照耀中国》的书；它研究的是：在斯诺《红星》出现之前，毛泽东在中国、苏联、欧美、日本的报刊、图片、传记和情报系统中如何被误认、想象、宣传，后来又如何被《红星》重新塑造成“可见的红色领袖”。

我读下来觉得，本书真正的主角不是毛泽东本人，而是“形象如何产生”。毛泽东当然在场，斯诺当然在场，但作者更关心的是：照片如何流通，报道如何被剪裁，翻译如何改变意义，政治组织如何选择展示或隐藏信息。换句话说，这是一本关于“现代政治人物如何被媒体和史料制造出来”的书。

全书可以分成三大块：第一块是《红星》之前的毛泽东形象，重点在早期传记、肖像、共产国际和日本情报；第二块是斯诺如何进入陕北、如何完成《红星》；第三块是《红星》出版后的版本、翻译、政治命运和后世阅读。

这本书读起来像学术侦探小说：从一张错照出发，沿着剪报、照片、档案、版本、译文一路追踪。它不是用宏大理论压人，而是用很多小证据迫使我们改变想法。

## 二、核心结论：这本书到底讲了什么

| 核心结论 | 我的转述 |
| --- | --- |
| 1. 《红星》不是单纯报道，而是一次形象生成事件 | 它让毛泽东从模糊传闻变成有脸、有声音、有故事的人；照片、第一人称自述和长征叙事共同塑造了“可被世界理解的中共”。 |
| 2. 1936年前的毛泽东并不“清晰可见” | 外界掌握的是敌对报道、小报传闻、共产国际材料、剪裁照片和二手传记。今天看起来荒唐的误认，在当时有其信息环境。 |
| 3. 毛泽东形象的传播不完全由毛本人控制 | 莫斯科、王明、欧美左翼刊物、日本中共问题专家、斯诺夫妇、中文译者都参与了这个形象的生产。 |
| 4. 王明并非只能被理解为“反毛” | 至少在1934年前后，王明等莫斯科中共代表也曾以毛泽东作为中国苏维埃运动象征来进行对外宣传。 |
| 5. 波多野事件显示“信息错误”可能是政治选择 | 日本外务省情报部很可能知道斯诺已提供真实照片，却仍使用“胖子毛泽东”错照，以避免真实毛泽东形象带来的同情效果。 |
| 6. 斯诺既有职业坚持，也有情感靠近 | 他拒绝把全书交给中共审查，保留第一人称自述与大量现场观察；但他和采访对象之间确有亲近关系，对中共有明显同情。 |
| 7. 笔录确认不等于审查 | 毛泽东确认通过翻译形成的采访记录，是跨语言采访的准确性要求；把这直接说成“毛审查了全书”，证据不足。 |
| 8. 《红星》的生命史充满矛盾 | 它帮助中共获得巨大声望，却在新中国长期被内部化、限制流通，因为原书并不完全符合后来官方史观。 |
| 9. 版本差异本身就是历史材料 | 英文版、中文版、俄文版、日文版的删改与重排，显示不同政治系统如何消化同一文本。 |
| 10. 本书真正的方法是“回到不知道结局的时刻” | 作者不是从胜利后的毛泽东倒推，而是重建1936年前外界如何在碎片中想象毛泽东。 |

## 三、章节整理：这本书怎样展开论证

### 序章：毛泽东照片之谜

全书从一张荒唐的错照开始。1937年8月，日本政府公报附录《周报》刊登一张所谓“毛泽东”照片，画面上的人却像阔佬、公司老板，完全不像后来被世界熟知的毛泽东。作者没有把这当笑话，而是把它当成进入历史现场的钥匙：如果当时连日本外务省情报部门都可能把错人当成毛泽东，那说明1936年前后的毛泽东形象并不是透明的、稳定的，而是在信息封锁、政治宣传、报刊转载和传闻误差中被拼凑出来的。

### 这一章的重点可以这样抓：

作者真正追问的不是“这张照片为什么错”，而是“在《红星照耀中国》出现之前，世界到底怎样认识毛泽东？”

序章强调一个反直觉事实：我们今天觉得错照可笑，是因为我们知道结局；当时许多中国人、日本人、外国记者都不知道毛泽东的真实容貌。

斯诺的意义由此凸显：他不只是写了一本畅销书，而是用照片、采访和毛泽东自述，为世界提供了第一套相对稳定、可视化、可传播的“毛泽东形象”。

### 我的理解

我的理解：序章最有价值的地方，是提醒我不要用后见之明嘲笑历史中的人。今天看来显而易见的东西，在当时可能根本不可见。历史研究的第一步，恰恰是把自己从“我早就知道结局”的位置上撤下来。

### 第一章：不为人知的革命家

这一章先补足毛泽东1949年前的基本经历，但它不是普通毛传。作者的重点是：即便毛泽东后来成为中国革命的最高象征，在1930年代前半期，他对外界来说仍然是模糊、零碎、甚至边缘的。国民党封锁共产党的信息，共产党自身也不急于宣传领袖个人；共产国际虽影响中共，却并不真的掌握毛泽东的具体情况。

### 这一章的重点可以这样抓：

毛泽东从湖南农村出身、新式教育、五四运动、参与中共一大、国共合作、农民运动、井冈山与朱德会合、瑞金苏维埃、长征与遵义会议等经历，被作者概括为“后来形象”的历史素材。

国内早期关于毛泽东的报道和传记多来自敌对阵营、政治小报、名人录等材料，既有真实片段，也有猜测和偏见。

共产国际对毛泽东的认识也有限，这一点打破了“莫斯科无所不知、无所不控”的简单想象。

### 我的理解

我的理解：这一章告诉我，伟人形象不是从一开始就完整存在的。后来被写成“必然上升”的人物，在当时很可能只是许多革命者之一。所谓历史人物的“必然性”，往往是在胜利以后被倒推出来的。

### 第二章：肖像与真实人物

第二章进入视觉材料。1930年代前半期，欧美左翼刊物、苏联出版物和共产国际系统开始出现毛泽东肖像，但大多来源单一、质量粗糙、判断混乱。作者反复追踪这些肖像的来源，发现许多都可追溯到1927年武汉时期的一张国民党干部合影，后来被裁切、临摹、改绘、印刷、再传播。

### 这一章的重点可以这样抓：

《今日中国》等左翼刊物曾刊出“其貌不扬”的毛泽东素描；这种图像并不一定像毛，却因为缺少替代资料而进入传播链。

共产国际曾出现毛泽东“病逝”之类错误信息，说明跨国革命网络中的情报并不可靠。

俄国作家爱伦堡等人写出的早期毛泽东传记，既有革命文学色彩，也有资料不足造成的想象。

### 我的理解

我的理解：这一章像是在看一张图像如何被反复复印。每复印一次，轮廓更模糊，但权威感反而可能更强。只要刊物、组织和意识形态足够有声量，错误也能暂时成为“资料”。

### 第三章：国际共产主义舞台上不见身影的著名革命家

这一章非常有意思：毛泽东在中国根据地内的政治处境一度并不顺利，被党内对手排挤；但在莫斯科和国际宣传中，他却被逐渐塑造成中国苏维埃运动的代表人物。作者特别指出，王明在1934年前后反而曾经推动毛泽东著作和形象的对外宣传，这与后来“王明压制毛泽东”的固定叙事并不完全一致。

### 这一章的重点可以这样抓：

莫斯科出版的《只有苏维埃能够救中国》《经济建设与查田运动》等小册子，是早期以毛泽东为核心的宣传材料；后者可视为非常早的毛泽东著作集。

哈马丹、高自立等人写的毛泽东传记，说明早期毛传并非一条单线发展，而是在莫斯科、中国、报刊、党内外材料之间互相借用。

高自立等中共相关人士也在使用外部小报材料，说明信息来源并不总是“党内权威文献”，而是复杂混杂的。

### 我的理解

我的理解：这一章最能纠正单线党史叙事。政治人物的形象不一定由其本人控制，也不一定与其真实权力同步。一个人在组织内部可能受挫，在外部宣传中却可能被推到台前。

### 第四章：“胖子毛泽东”照片之谜

这一章是全书最像侦探小说的部分。作者追踪《周报》那张“胖子毛泽东”照片的来历，发现它在山本实彦《支那》中已经出现，后来被日本外务省情报部公报采用。进一步追查后，作者把关键人物锁定为波多野乾一：他是战前日本有名的中共问题专家，曾在外务省情报部从事情报分析。

### 这一章的重点可以这样抓：

波多野并非外行，他熟悉中共资料，甚至早已注意到有两种相互矛盾的“毛泽东”照片：一种瘦削、高颧骨，一种胖乎乎像公司老板。

1937年日本媒体已经能看到斯诺拍摄的真实毛泽东照片，但外务省公报仍使用“胖子”照片，作者因此推断：这很可能不是单纯疏忽，而有信息选择和舆论操纵的成分。

“胖子”本人到底是谁，作者没有武断下结论。他提出冯玉祥、李杜等可能性，但都缺乏足够证据，因此保留悬案。

### 我的理解

我的理解：这一章最重要的不是破案，而是展示史料判断的边界。好的历史写作不是把所有谜团都讲成“真相大白”，而是在证据够的时候推进，证据不够的时候停住。

### 第五章：斯诺前往“红色中国”

第五章把焦点转向斯诺如何进入陕北。传统读法容易把这讲成“孤胆记者闯入赤匪巢穴”，但作者通过海德姆、宋庆龄、史沫特莱、冯雪峰、刘鼎、董健吾等人的线索，说明斯诺之行其实依赖复杂的地下交通、统战网络和政治时机。

### 这一章的重点可以这样抓：

斯诺并非独自行动，同行者有乔治·海德姆；海德姆后来留在陕北并成为马海德。

斯诺第一次尝试进入陕北未成功；第二次成功，与张学良和中共之间关系缓和、地下交通线形成有关。

刘鼎负责接送，冯雪峰在上海传递信息，宋庆龄、史沫特莱等人也发挥作用；这些人后来很多因为政治环境变化而长期不便被公开提及。

关于《红星》是否经过毛泽东审查，作者的判断很细：斯诺确实把采访记录译回中文请毛确认，这是为了跨语言采访的准确性，不能简单等同于审查；但中共方面也确实曾因政治形势变化要求删除或推迟发表部分内容。

### 我的理解

我的理解：这一章把“新闻采访”还原为政治现场。斯诺既不是完全中立的旁观者，也不是被操纵的传声筒。他进入陕北的过程，本身就是一场涉及信任、交通、保密、统战和叙事控制的合作。

### 第六章：“红星”终于升起

第六章讨论《红星照耀中国》的诞生、版本变化和在中国的命运。斯诺1936年10月回到北平后采访笔记、照片和影片，随后文章先在报刊杂志上产生影响，1937年英国版、1938年美国版出版后迅速畅销。它第一次让毛泽东和中共以照片、自述、长征故事、根据地生活的方式进入世界视野。

### 这一章的重点可以这样抓：

《红星》受到大量赞扬，也受到左翼内部批评；斯诺不是共产党员，他对共产国际、斯大林和统一战线的一些看法，让部分左翼人士不满。

英文版经历1938、1944、1968等不同阶段的修改。斯诺面对后来的历史变化，并未完全重写原书，而是尽量保留1936年现场感。

中文传播非常重要。《西行漫记》影响大量青年走向延安或同情中共，但1949年后它反而长期未公开再版，1960年复刊也属于内部读物。

原因之一是斯诺后来的一些判断与新中国官方叙事不合；另一个原因是《红星》中的人物形象和逸事，与后来形成的正式革命史、领袖叙事存在偏差。

1979年董乐山译本和吴黎平本出现，标志着《红星》在改革开放语境下重新被公开放回历史记忆中。

### 我的理解

我的理解：这章让我意识到，一本书的命运并不由它的“贡献”决定。它可以在革命前被需要，在革命胜利后被尴尬地隐藏。政治宣传会利用文本，也会在文本不再服从时搁置文本。

### 第七章：《红星》在苏联和日本

最后一章把《红星》的传播放到苏联和日本两个语境中看。苏联本来是国际共产主义宣传中心，但斯诺的《红星》反而成了竞争者，因为它不是从莫斯科生产出来的宣传品，而且包含一些不合苏联口径的内容。日本方面则更复杂：战前战时既翻译、介绍，又出于反共和战争需要而过滤、改写。

### 这一章的重点可以这样抓：

苏联最早对《红星》的介绍采取摘译方式，删去涉及党内问题、共产国际责任、斯大林路线等敏感内容。俄文单行本也大幅压缩改造，几乎成了另编宣传册。

战前日本已有人关注、翻译斯诺报道，但政府和媒体环境中的反共、战争、审查机制，使《红星》的真实影响受到筛选。

战后日本重新出版和阅读《红星》，也经历从左翼热情到学术化、历史化的变化。

### 我的理解

我的理解：同一本书进入不同政治系统后，会被重新制造。翻译不是单纯语言转换，而是政治过滤、叙事重排和意义再生产。

结语与中文版后记：怎样重新读《红星》

作者在结语中承认，《红星》今天的吸引力已不同于1930年代。它曾经被费正清等人称为经典，也曾激励许多青年；但经历毛泽东时代、文革、改革开放之后，读者会带着新的疑问重新面对它。作者并不要求我们恢复过去的热情，而是建议把《红星》当作中国现代史、中共史、毛泽东形象史和新闻史的直接材料来重读。

### 这一章的重点可以这样抓：

中文版后记说明，本书不是要再写一部毛传，而是研究“成为著名人物之前的毛泽东”在同时代人心中的样子。

作者强调自己的优势反而来自“不熟悉”：中国读者太熟悉后来毛泽东的形象，容易被后见之明束缚；外国学者从资料碎片出发，反而能更接近当时人的迷茫状态。

全书采用侦探小说式叙述，是为了让看似琐碎的照片、剪报、版本差异、译文删节，重新显出历史意义。

### 我的理解

我的理解：这本书最打动我的不是某个结论，而是一种历史意识：越熟悉的东西越可能看不见。重新理解毛泽东、斯诺和《红星》，不是因为我们不知道他们后来怎样，而是要暂时忘掉后来，回到当时人并不知道结局的现场。

| 毛泽东 | 本书的研究对象不是“晚年领袖”，而是1936年前后仍被外界误认、猜测、神秘化的革命者。 |
| 埃德加·斯诺 | 《红星照耀中国》作者。他通过陕北采访、照片和毛泽东自述，改变了毛与中共的国际形象。 |
| 尼姆·韦尔斯 | 斯诺妻子与合作者，后赴陕北采访；《红星》部分照片和信息更新与她有关。 |
| 乔治·海德姆/马海德 | 与斯诺同行进入陕北，后来留在中共根据地，成为新中国医疗卫生事业中的重要外籍人士。 |
| 史沫特莱 | 帮助建立斯诺与中共之间联系的外国左翼记者之一。 |
| 宋庆龄 | 为斯诺进入陕北提供重要支持，但相关角色长期不便公开。 |
| 冯雪峰 | 上海地下交通和文化界联络中的关键人物，也牵涉“鲁迅火腿”等细节。 |
| 刘鼎 | 在西安与陕北之间负责接应斯诺等人的交通安排，后来因政治环境长期沉默。 |
| 董健吾 | 《红星》中“王牧师”的真实身份，秘密护送斯诺进入陕北。 |
| 王明 | 一般被记为“压制毛泽东”的人物，但本书显示其在莫斯科宣传中曾推动毛形象。 |
| 波多野乾一 | 日本中共问题专家，作者认为他与“胖子毛泽东”照片在《周报》上的使用关系重大。 |
| 董乐山 | 1979年《西行漫记》重译本译者，其译本让《红星》重新进入公开阅读。 |
| 吴亮平/吴黎平 | 斯诺采访毛泽东时的重要翻译，后出版《毛泽东1936年同斯诺的谈话》。 |

## 五、时间线：一个政治形象怎样形成

| 时间 | 关键变化 |
| --- | --- |
| 1927 | 毛泽东在国共合作、农民运动、国民党干部合影等历史场景中出现；后来许多早期肖像可追溯到这一时期照片。 |
| 1930 | 红军攻占长沙等事件让国外媒体开始关注中国共产党和红军。 |
| 1931 | 中华苏维埃共和国在瑞金成立，毛泽东任政府主席；外界仍难以接触真实信息。 |
| 1934 | 莫斯科出版早期毛泽东著作/报告材料；苏联和欧美左翼刊物出现毛泽东肖像与传记。 |
| 1935 | 长征途中遵义会议后，毛泽东在中共内部重新取得关键地位。 |
| 1936春 | 斯诺、海德姆尝试进入陕北未果；上海、陕北、西安之间的秘密联系逐渐建立。 |
| 1936夏秋 | 斯诺进入陕北采访约三个月，拍摄照片、记录毛泽东自述和根据地生活。 |
| 1936末-1937 | 斯诺采访文章和照片先于单行本在报刊杂志传播，真实毛泽东图像开始取代旧图像。 |
| 1937-1938 | 《红星照耀中国》英美版出版并轰动；中文《西行漫记》出版并影响大量中国青年。 |
| 1949以后 | 《西行漫记》在新中国长期未能公开流通，1960年复刊也多为内部读物。 |
| 1968 | 斯诺出版扩充修订版，增加附录和照片，但仍尽量保留早期报道形态。 |
| 1979 | 董乐山译本与吴亮平相关本出版，《红星》重新公开进入中文读者视野。 |

## 六、版本、翻译与传播：为什么“红星”会变形

这本书反复提醒我：一本书不是出版后就固定不动。尤其《红星》这种政治性、新闻性、历史性都很强的文本，每进入一种语言、一个国家、一个时期，都会被重新解释。

| 版本/语境 | 意义 |
| --- | --- |
| 英国/美国早期版 | 确立《红星》的国际声誉。它把毛泽东自述、长征故事、红军生活、照片等组合起来，形成对外界极具吸引力的新叙事。 |
| 1938年中文《西行漫记》 | 对中国青年影响巨大。许多人不是通过中共正式文件，而是通过这本带有纪实魅力的书对延安和中共产生想象。 |
| 苏联俄文版 | 不是简单翻译，而是按照苏联政治口径大幅删节和重组，弱化与共产国际、斯大林有关的争议。 |
| 日本战前/战时译介 | 既有知识界对中共的兴趣，也有政府反共、战争和审查带来的过滤。日本不是不知道斯诺，却不愿完全接受斯诺展示的毛泽东。 |
| 新中国早期 | 《西行漫记》贡献巨大，却长期未公开流通。它与后来官方化的革命史、领袖叙事存在不便之处。 |
| 1979年董乐山译本 | 在改革开放和中美关系正常化背景下重新推出，成为后来最通行的中文阅读入口。 |

我的判断

《红星》的“变形”不是偶然错误，而是现代政治文本的常态。每一种版本都暴露了它所在政治环境的需要：苏联要统一战线与斯大林权威，日本要反共与战争动员，新中国要正式党史与领袖形象，改革开放后则需要重新连接世界与历史。

## 七、我认为最值得保留的部分

| 我认为应保留的重点 | 为什么重要 |
| --- | --- |
| 1. “回到不知道结局的现场” | 作者一直在反抗后见之明。我们今天知道毛泽东是谁，所以觉得错照荒唐；但历史研究恰恰要问：当时的人为什么不知道？他们能知道什么？不能知道什么？ |
| 2. 史料批判比立场判断更重要 | 这本书没有急着表态毛泽东好坏、斯诺好坏，而是先问材料从哪里来，谁复制了谁，谁删掉了什么，谁为什么隐藏某些信息。 |
| 3. 对“审查说”的细分 | 确认采访笔录、请求删除敏感材料、审查全书，是三件不同的事。作者没有替斯诺洗白，而是把不同层级分开。 |
| 4. 对“宣传”的复杂理解 | 《红星》确实帮了中共，但不等于它只是宣传。宣传、新闻、同情、现场记录和历史偶然，可以同时存在于一个文本中。 |
| 5. “贡献者后来被隐藏”的悖论 | 冯雪峰、刘鼎、董健吾、宋庆龄、韦尔斯等人在《红星》形成中作用很大，但因地下工作、政治运动、官方叙事等原因，长期不能完整出场。 |

## 八、容易误读的地方

| 容易误读 | 笔记里的纠正 |
| --- | --- |
| 误读一：这本书是在否定《红星》的价值 | 不对。作者承认《红星》作为现场报道和历史事件的价值，只是要求我们不再把它当成透明、完整、无问题的真相。 |
| 误读二：斯诺只是中共宣传工具 | 过于简单。斯诺确实同情中共，也被陕北经验打动，但他同时坚持记者身份，对共产国际、斯大林路线和中共一些要求并不完全顺从。 |
| 误读三：毛泽东审查了《红星》 | 作者区分“确认采访记录”和“审查全书”。前者存在，且有合理性；后者缺乏充分证据。中共方面曾要求删改个别人物生平材料，但这与毛对全书进行审查不是一回事。 |
| 误读四：日本刊错照片只是无能 | 不一定。波多野作为专家不太可能完全没看到斯诺照片。作者更倾向于认为，错照的使用包含有意隐藏真实照片、操纵读者印象的因素。 |
| 误读五：毛泽东早期形象从一开始就围绕他本人展开 | 不是。早期传播链条里，朱德、周恩来、共产国际、王明、苏维埃运动、抗日统一战线等都在场，毛的中心地位是逐渐被建构出来的。 |

## 九、批判性评价与我的保留意见

我觉得这本书最大的优点，是把毛泽东研究从宏大判断拉回到具体材料。很多历史讨论太快进入立场：喜欢毛的人把《红星》当见证，反感毛的人把《红星》当受骗。本书的处理方式更稳：先看材料如何形成，再看判断能走多远。

第二个优点是叙述非常有耐心。作者愿意花很长篇幅追踪一张照片、一篇剪报、一个译名、一本小册子的来龙去脉。这种写法表面琐碎，实际是在训练读者：历史并不是大词堆出来的，而是小证据累出来的。

但我也有保留。第一，本书对材料链条的追踪很精彩，但有时会让主线显得过细，尤其对不熟悉中共早期史、共产国际、日本报刊史的人来说，阅读负担比较大。所以我的笔记特意把它成“问题—证据—结论”的结构。

第二，作者对“胖子毛泽东”照片的追查很有说服力，但最后仍有未解之处：照片上的人到底是谁，尚无定论。这里不能为了满足故事感而硬说破案。

第三，本书重在形象生成和资料传播，因此对毛泽东思想本身、红军社会结构、延安政治实践等问题不会展开太多。如果想理解中共制度和意识形态，还需要配合其他书读。

我自己的总评

这本书最适合当成“如何读历史材料”的范例。它让我看到，一个政治形象的诞生不只是领袖个人魅力的结果，而是照片、文字、翻译、情报、审查、传播网络和历史偶然共同作用的结果。

## 十、最后留给自己的简记

本书研究的不是毛泽东的一生，而是1936年前后毛泽东早期形象如何被世界认识、误认、重塑。

《红星》之前，外界关于毛泽东的材料非常混乱：有小报传记、共产国际宣传、错误肖像、剪裁合影、甚至“胖子毛泽东”错照。

斯诺的核心贡献，是把毛泽东从传闻中变成“可见的人”：有照片、有自述、有长征故事、有根据地现场。

《红星》不是纯粹中立报道，但也不能简单说成宣传。它是同情性新闻、政治传播和现场见证的混合体。

毛泽东确认斯诺的采访记录，主要是为解决翻译和细节准确问题；这与“审查全书”不是一回事。

日本外务省使用错照，很可能不是单纯无知，而是有选择地隐藏斯诺所提供的真实且更有吸引力的毛泽东形象。

《红星》帮助中共获得声望，却在新中国长期不便公开流通，因为它与后来的正式革命史和领袖叙事并不完全一致。

这本书最重要的方法，是让读者暂时忘记结局，回到当时人无法预知结果、只能在碎片中判断的状态。

可用于讨论/写作的题目

为什么说“错误照片”比正确照片更能帮助我们理解1930年代的信息环境？

如果斯诺没有进入陕北，毛泽东和中共的国际形象会不会以完全不同的方式形成？

《红星》既有同情又有报道价值，我们该如何评价这种“带立场的真实”？

当一个文本被不同国家反复删改、摘译、重译时，原作还存在吗？

为什么革命胜利以后，曾经帮助革命宣传的文本反而可能变得尴尬？

最后的个人总结

如果只把《红星照耀中国》看成一本曾经“宣传中共”的书，就会把问题看窄；如果只把它当成“真实记录”，也会把问题看浅。石川祯浩这本书真正让我看到的是：历史人物的形象不是自然出现的，而是在信息匮乏、政治需要、媒体技术、翻译删改和个人冒险之间形成的。

所以，“红星是怎样升起的”这个问题，不只是问毛泽东如何成名，也是在问现代人如何通过照片、书、报刊、版本和故事认识政治。毛泽东后来太有名，反而遮住了一个更基本的问题：在他还没有成为世界熟知的毛泽东之前，人们到底看见了什么，又看错了什么？这本书的价值就在于，它让那些被胜利叙事覆盖的错看、误读、半真半假的材料重新浮出水面。
